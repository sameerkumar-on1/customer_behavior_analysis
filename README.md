
# 📊 Data Analytics Project

## Overview
This project demonstrates an end-to-end data analytics workflow, from data collection and cleaning to visualization and reporting. The goal is to transform raw data into meaningful insights using industry-standard tools and techniques.

The project includes:
- Data loading and preprocessing using Python
- Exploratory Data Analysis (EDA)
- Data cleaning and transformation
- SQL analysis using PostgreSQL
- Interactive Power BI dashboard
- Business insights report
- Presentation created with Gamma

---

## Dataset

- **Source:** Kaggle
- **Format:** CSV
- **Description:** The dataset contains customer demographics, purchasing behavior, product preferences, and transaction details, enabling analysis of shopping patterns and customer trends.

- **Dataset:** Customer Behavior Analysis (Kaggle)

## Project Workflow

### 1. Data Loading
- Imported the dataset into Python.
- Inspected data structure and data types.

### 2. Exploratory Data Analysis (EDA)
- Analyzed distributions and trends.
- Identified missing values and outliers.
- Created visualizations to understand customer behavior.

### 3. Data Cleaning
- Removed duplicate records.
- Handled missing values.
- Corrected data types.

### 4. SQL Analysis
- Imported the cleaned dataset into PostgreSQL.
- Wrote SQL queries to answer business questions.

### 5. Dashboard Development
- Built an interactive Power BI dashboard.
- Added KPIs, charts, and slicers.

### 6. Reporting
- Documented key findings.
- Created a presentation using Gamma.
